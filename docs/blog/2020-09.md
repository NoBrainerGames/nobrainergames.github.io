<sup>First 3D scene in Nim.</sup>

This scene -- which was based on Diligent Engine's [Shadow sample](https://github.com/DiligentGraphics/DiligentSamples/tree/master/Samples/Shadows) -- proved that I could create interactive 3D scenes using Diligent Engine using my cinterop library.

At this point the code to load [GLTF](https://www.khronos.org/gltf/) models had taken shape, as that was required to load the entire scene in the first place. Diligent Engine already profiled an [asset loader](https://github.com/DiligentGraphics/DiligentTools/tree/master/AssetLoader) imple

It was also at this point that I decided to stick with [Vulkan](https://www.vulkan.org/) as my primary rendering target due to its ubiquity. It took a degree of fiddling since the original sample used DirectX, but thankfully the API abstracted away the differences between rendering targets well enough to let me figure things out quickly.

Now that I had a scene, the next frontier was being able to interact with it.
